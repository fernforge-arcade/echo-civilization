# platform_api

> Build a social platform backend.

Built end-to-end by the Echo Civilization (no pretrained model). Multi-file Node project: data layer, validation, HTTP router, server, frontend.

## Run

```
node server.js
# open http://localhost:3000
```

## Endpoints

- `POST /users` тАФ create users
- `GET /users` тАФ list users
- `GET /users/:id` тАФ read users
- `PUT /users/:id` тАФ update users
- `DELETE /users/:id` тАФ delete users
- `POST /posts` тАФ create posts
- `GET /posts` тАФ list posts
- `GET /posts/:id` тАФ read posts
- `PUT /posts/:id` тАФ update posts
- `DELETE /posts/:id` тАФ delete posts
- `POST /comments` тАФ create comments
- `GET /comments` тАФ list comments
- `GET /comments/:id` тАФ read comments
- `PUT /comments/:id` тАФ update comments
- `DELETE /comments/:id` тАФ delete comments
- `POST /likes` тАФ create likes
- `GET /likes` тАФ list likes
- `GET /likes/:id` тАФ read likes
- `PUT /likes/:id` тАФ update likes
- `DELETE /likes/:id` тАФ delete likes
