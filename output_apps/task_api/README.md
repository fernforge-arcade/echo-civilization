# task_api

> Build a task tracker API.

Built end-to-end by the Echo Civilization (no pretrained model). Multi-file Node project: data layer, validation, HTTP router, server, frontend.

## Run

```
node server.js
# open http://localhost:3000
```

## Endpoints

- `POST /tasks` — create tasks
- `GET /tasks` — list tasks
- `GET /tasks/:id` — read tasks
- `PUT /tasks/:id` — update tasks
- `DELETE /tasks/:id` — delete tasks
