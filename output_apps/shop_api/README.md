# shop_api

> Build a shop backend.

Built end-to-end by the Echo Civilization (no pretrained model). Multi-file Node project: data layer, validation, HTTP router, server, frontend.

## Run

```
node server.js
# open http://localhost:3000
```

## Endpoints

- `POST /users` тАФ create users
- `GET /users` тАФ list users
- `GET /users/:id` тАФ read users
- `PUT /users/:id` тАФ update users
- `DELETE /users/:id` тАФ delete users
- `POST /products` тАФ create products
- `GET /products` тАФ list products
- `GET /products/:id` тАФ read products
- `PUT /products/:id` тАФ update products
- `DELETE /products/:id` тАФ delete products
- `POST /orders` тАФ create orders
- `GET /orders` тАФ list orders
- `GET /orders/:id` тАФ read orders
- `PUT /orders/:id` тАФ update orders
- `DELETE /orders/:id` тАФ delete orders
