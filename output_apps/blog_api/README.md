# blog_api

> Build a blog backend.

Built end-to-end by the Echo Civilization (no pretrained model). Multi-file Node project: data layer, validation, HTTP router, server, frontend.

## Run

```
node server.js
# open http://localhost:3000
```

## Endpoints

- `POST /posts` — create posts
- `GET /posts` — list posts
- `GET /posts/:id` — read posts
- `PUT /posts/:id` — update posts
- `DELETE /posts/:id` — delete posts
- `POST /comments` — create comments
- `GET /comments` — list comments
- `GET /comments/:id` — read comments
- `PUT /comments/:id` — update comments
- `DELETE /comments/:id` — delete comments
